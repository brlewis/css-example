To make text on your page easier to read, enclose all the content in a single div that is centered and has a maximum width. It helps to have at least a basic understanding of the box model.

The current fashion in web design is for images to be placed without any borders or shadows. But if this isn't the style you want, there are options.

You can put a little border around your photo by specifying the CSS properties for border and padding.

You can give photos a friendly look with rounded corners using border-radius. Set its value high enough and you have a circle.

You can add a 3D effect with box-shadow.

Don't overdo it. Pick just enough decoration and stop.
